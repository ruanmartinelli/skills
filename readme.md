# ruanmartinelli/skills

These are my agent skills. There are many like them, but these are mine.

[![skills.sh](https://skills.sh/b/ruanmartinelli/skills)](https://skills.sh/ruanmartinelli/skills)

## Install

```bash
npx skills add ruanmartinelli/skills
```

Or install a single skill:

```bash
npx skills add ruanmartinelli/skills --skill <name>
```

## Skills

| Skill                     | Command                      | Description                                                           |
| ------------------------- | ---------------------------- | --------------------------------------------------------------------- |
| architect                 | `/architect`                 | Plan before coding: file tree, ASCII diagram, function signatures.    |
| lazy                      | `/lazy`                      | Bias toward deletion and the smallest change that solves the problem. |
| typescript-best-practices | `/typescript-best-practices` | TypeScript best practices for `.ts`/`.tsx` files.                     |
| unslop                    | `/unslop`                    | Cut AI tells from any writing, add human voice.                       |
